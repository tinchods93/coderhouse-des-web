aws-file-handler

Repositorio
Descripción

Esta es una plantilla para manejar archivos en AWS. La carga de imagenes se hace hacia una cuenta en cloudinary y su vez la optimizamos. Mientras que, la metadata de las mismas se guardan en una tabla de DynamoDB.
Mientras que los archivos como PDF, videos, y otros, se guardan directo en una S3.

Funciones

  • * CRUD Imagenes aws+cloudinary

  • * CRUD Archivos aws s3

Caracteristicas

  • * Infraestructura como código mediante Serverless Framework

  • * Arquitectura Hexagonal

  • * Inyeccion de dependencias

  • * Test Driven Development

  • * Cloudinary

cloudformation serverless jest tsyringe typescript

aws-custom-authorizer

Repositorio
Descripción

Esta es una plantilla que actua como autorizador para Api Gateway.
Internamente consume un servicio de autenticacion de tokens como aws-token-security.

Funciones

  • * Validacion de Token

  • * Validacion de Scopes

  • * Permitir o no, que una Request HTTP ejecute una Lambda

Caracteristicas

  • * Infraestructura como código mediante Serverless Framework
